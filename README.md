## 🛠 Features Implemented
According to the Midterm requirements, this project includes:

* **5 Semantic Pages:** Home, Artists, Albums, Gallery, and Contact.
* **Modern Layouts:** * **Flexbox** for a responsive navigation bar and header.
    * **CSS Grid** for the artist card gallery and music widgets.
* **Responsive Design:** Fully adaptive for Desktop, Tablet, and Mobile using Media Queries.
* **Interactivity:** Hover effects on cards, sticky navigation, and styled form elements.
* **Data Representation:** A styled HTML table for album comparisons.
* **External Assets:** Integrated Google Fonts (Poppins) and 10 interactive Spotify Widgets.

## 📂 Project Structure
* `index.html` - Homepage
* `artists.html` - List of 10 featured artists
* `albums.html` - Album comparison table
* `gallery.html` - Spotify music players
* `contact.html` - Feedback & request form
* `/css/style.css` - External stylesheet with CSS Variables
* `/img/` - Directory for artist photography
